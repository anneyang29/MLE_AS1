End-to-End Data Pipeline for Credit Risk Modeling

Short Summary

Built a scalable PySpark ETL pipeline based on the Medallion Architecture (Bronze–Silver–Gold) to transform raw credit application data into clean, analysis-ready feature tables for downstream machine learning.

Overview

This project focuses on building a production-ready data pipeline for credit risk modeling. The pipeline automates data ingestion, cleaning, transformation, and feature engineering using a structured data engineering workflow. It supports batch processing, improves data quality, and produces Gold-layer datasets that can be used for model training, evaluation, and monitoring.

Key Highlights:

- Designed a Bronze–Silver–Gold ETL workflow using PySpark
- Processed raw credit application data into validated feature tables
- Applied cleaning, transformation, and feature engineering for modeling readiness
- Produced analysis-ready datasets for downstream machine learning tasks
